Tentang Proyek

Beranda Prediksi Analisis Data Tentang

Deskripsi singkat

Proyek ini mengembangkan model machine learning untuk memperkirakan risiko serangan jantung menggunakan dataset terbuka. Tujuan: pendidikan dan eksperimen analisis data.

Peringatan

Aplikasi ini bukan alat diagnosis klinis. Hasil prediksi tidak dapat menggantikan pemeriksaan dokter.

Struktur proyek

  1. notebooks/ — Notebook Jupyter (CRISP-DM: Business → Modeling → Visualization).
  2. data/ — CSV dataset.
  3. models/ — Model tersimpan (pickle).
  4. templates/ & static/ — Frontend Flask.

Cara menjalankan (singkat)

pip install -r requirements.txt
python app.py
# buka http://127.0.0.1:5000
