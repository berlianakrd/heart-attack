About This Project

AI-powered Heart Attack Risk Prediction System untuk Populasi Indonesia

Project Overview

Indonesia Heart Attack Prediction adalah sistem prediksi berbasis machine learning yang dirancang untuk membantu identifikasi risiko serangan jantung pada populasi Indonesia.

Sistem ini menggunakan pendekatan Data Science Life Cycle yang komprehensif, mulai dari business understanding hingga deployment, dengan mengimplementasikan algoritma supervised learning untuk klasifikasi.

High Accuracy
Model accuracy >85%
Comprehensive Data
28 health features analyzed
Indonesia-Specific
Tailored for Indonesian population
Real-time Prediction
Instant risk assessment

Data Science Life Cycle Implementation

1
Business Understanding

Mendefinisikan problem statement, objectives, dan success criteria untuk prediksi serangan jantung di Indonesia. Mengidentifikasi stakeholders dan kebutuhan bisnis.

2
Data Mining

Mengumpulkan dan mengekstrak dataset dengan 500+ sampel dan 28 features komprehensif mencakup demografis, klinis, gaya hidup, dan faktor lingkungan.

3
Data Cleaning

Membersihkan data dari missing values, duplicates, outliers, dan inkonsistensi. Memastikan kualitas data >99% untuk modeling yang optimal.

4
Data Exploration

Exploratory Data Analysis (EDA) untuk memahami distribusi data, korelasi antar variabel, dan mengidentifikasi patterns serta insights penting.

5
Feature Engineering

Membuat new features (risk score, age groups, BP categories), encoding categorical variables, feature selection, dan scaling untuk improve model performance.

6
Predictive Modeling

Training multiple classification models (Logistic Regression, Decision Tree, KNN), hyperparameter tuning, cross-validation, dan model evaluation.

7
Data Visualization & Deployment

Membuat comprehensive visualizations, generating insights, dan deploy model sebagai web application untuk akses real-time prediction oleh users.
